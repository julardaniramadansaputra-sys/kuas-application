Physics-Informed AI Framework for Predictive Maintenance

Focus: Power Electronics in Renewable Energy Systems

Project Summary

As renewable energy adoption accelerates, the unexpected failure of power inverters remains a significant bottleneck for system efficiency. Purely physics-based models are often too complex for real-time monitoring, while purely AI-driven models struggle when historical failure data is scarce. This research addresses this gap by developing a Physics-Informed AI Framework that combines the strengths of both domains.

My approach integrates domain knowledge—specifically physics-based degradation indicators—into a Long Short-Term Memory (LSTM) neural network. By feeding the AI model with theoretically grounded features rather than just raw sensor noise, the framework aims to improve the reliability of predictive maintenance. This method focuses on accurately forecasting component degradation to enable timely intervention.

Current Status (Preliminary)

Current validation involves rigorous simulations using PLECS software and standardized NASA datasets. Early tests show promising potential, achieving an 85% accuracy rate in predicting failure events with a 30-day lead time. This portfolio documents the development of this framework, detailing the methodology, preliminary simulation results, and the roadmap for future hardware-in-the-loop implementation.
