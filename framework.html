Proposed Framework

A Hybrid Architecture: Combining Physics-Based Features with Deep Learning.

System Architecture

The proposed framework consists of three distinct layers designed to transform raw sensor data into actionable maintenance decisions.

Layer 1: Physics-Based Indicators

Instead of feeding raw noisy signals directly into the AI, this layer utilizes domain knowledge to extract meaningful features. Key indicators include Collector-Emitter Voltage ($V_{ce}$) shifts and thermal impedance changes, which are direct physical symptoms of IGBT bond-wire fatigue.

Layer 2: AI Prediction (LSTM)

The extracted physics features serve as input for a Long Short-Term Memory (LSTM) network. LSTM was chosen for its ability to capture long-term temporal dependencies in time-series data, making it ideal for tracking the slow degradation process of power electronics.

Layer 3: Decision Support

The final layer outputs the Remaining Useful Life (RUL). A threshold-based logic is applied to categorize the component health status (Healthy, Warning, or Critical), providing clear maintenance recommendations to operators.
